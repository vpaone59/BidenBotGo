## Before running...
- Requires Go >= 1.22
- Create .env, add Discord bot token

## To run locally:
- Run: `go run main.go`

## To run locally with Air
- go install github.com/cosmtrek/air@latest
- Run: `air init` to create .air.toml
- Run: `air`

## To run with Docker
- Run: `docker compose up --build -d`
